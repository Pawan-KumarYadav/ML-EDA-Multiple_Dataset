# 🚀 Machine Learning EDA & Multiple Dataset Modeling Project

## 📌 Project Overview

This project demonstrates an end-to-end Machine Learning workflow including:

- Exploratory Data Analysis (EDA)
- Data Cleaning & Preprocessing
- Feature Engineering
- Model Building using Multiple ML Algorithms
- Model Evaluation & Comparison

The project is performed on multiple real-world and standard datasets such as the **Titanic Dataset** and **Iris Dataset**.

This repository shows practical implementation of the complete Data Science pipeline.

---

## 📊 Datasets Used

### 🧭 Titanic Dataset
- Survival prediction problem
- Binary Classification Task
- Handling missing values, encoding categorical variables
- Feature scaling and feature selection applied

### 🌸 Iris Dataset
- Multiclass Classification Problem
- Predicting flower species based on petal and sepal measurements
- Label Encoding applied on target variable
- Multiple classifiers compared
